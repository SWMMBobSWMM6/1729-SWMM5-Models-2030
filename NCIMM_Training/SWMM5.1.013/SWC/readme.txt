
Installing SWMM-CAT

SWMM-CAT runs as a desktop application on the Windows 7 or higher operating system.  It is distributed as a zipped file named swmm-cat1_0.zip (the 1_0 label will be updated as newer releases are made).  It can be downloaded from the following web site: 

http://www2.epa.gov/water-research/storm-water-management-model-swmm

The zip file contains three files, swmm-cat.exe, ZedGraph.dll, and this document. The files swmm-cat.exe and ZedGraph.dll should both be unzipped prior to launching the application and should reside in the same folder.  

You can launch SWMM-CAT independently of SWMM by double-clicking swmm-cat.exe in Windows Explorer or by creating a shortcut to it for your Start Menu.

SWMM-CAT can also run as a SWMM Add-in tool. See the SWMM-CAT User's Guide for instructions on how to register it as an add-in tool.
